# UniversityNotes
## Overview
 
This repository contains all coursework for my university projects and assignments, organized for easy access by teachers and instructors. It serves as a central hub where all submitted work can be reviewed at any time.
 
## Contents
 
The repository includes files across the following languages and technologies:
 
- **Java** — Object-oriented programming assignments and projects
- **HTML** — Web structure and markup files
- **CSS** — Styling and layout files
- **Python** — Scripting and programming assignments
## Purpose
 
This repository is maintained to give teachers convenient access to all submitted coursework in one place. Each file is organized by course or topic to make navigation straightforward.
 
## Repository Structure
 
```
/
├── Java/
│   └── (Java assignments and projects)
├── HTML/
│   └── (HTML files and web pages)
├── CSS/
│   └── (Stylesheets and design files)
└── Python/
    └── (Python scripts and assignments)
```
 
## How to Navigate
 
- Browse the folders above to find files by language or course.
- Each folder contains assignments relevant to that subject area.
- Files are named to reflect the assignment or topic they cover.
